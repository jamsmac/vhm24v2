# VenHubProdCalc Reference

This is a VendHub Production Calculator HTML file that uses:
- React 18 (from unpkg CDN)
- ReactDOM 18
- Babel standalone for JSX transformation
- Lucide React icons
- Recharts for charts
- TailwindCSS

## Structure
- Main component: `VendHubCalculator`
- External JSX file: `VenHubProdCalc.jsx`
- Uses createRoot from ReactDOM

## Purpose
Production calculator for vending machines - likely calculates:
- Ingredient consumption based on sales
- Expected vs actual stock levels
- Production planning

This should be integrated into the admin panel as a production/inventory calculator feature.
